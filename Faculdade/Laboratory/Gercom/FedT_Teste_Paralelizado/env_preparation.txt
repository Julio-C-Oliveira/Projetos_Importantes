01 - Instalar o Pip:
sudo apt-get update
sudo apt install python3-pip

02 - Instalar o Virtual Env:
sudo pip install virtualenv

03.1 - Criando um ambiente virtual:
virtualenv (nome do projeto sem os parenteses)

03.2 - Ativando o ambiente virtual:
source (nome do projeto sem os parenteses)/bin/activate

03.3 Opcional - Desativar o ambiente virtual:
deactivate

03.4 Opcional - Desfazer o ambiente virtual:
rm -r (nome do projeto sem os parenteses)
